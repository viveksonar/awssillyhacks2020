Silly C.I

Automated Jenkins Deployments using Alexa

How it works

Baes on jenkins open rest API.Silly C.I uses the jenkins API to send a POST request to the jenkins server and triggers the pipeline
What it is?

Silly C.I is a Alexa Business application that helps employees to triger the jenkins pipeline using Voice commands

Technologys/Tech stack used

The entire project comprises of Amazon ELB(to deploy the code),Amazon s3(to store the .war file generated),Alexa for business,AWS CLI,AWs EC2(to spin jenkins server)),Amazon

Is it safe any one can use it to trigger your builds!!!

Actually no since its a Alexa Business application it can only be used in the scope of organization
